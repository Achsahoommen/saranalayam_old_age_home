Saranalayam Old Age Home(renamed as Sharanstan)

A web-based donation and management system designed for saranalayam old age home to streamline donations, track daily records, and verify contributions.

🔗 Live Website:https://sharanstan.up.railway.app/

📖 Overview

Sharanstan is a Flask-based application built to improve the management of donations and daily operations in an old age home.

The system allows:

Donors to contribute funds easily
Administrators to maintain daily records

This project is designed for simplicity, transparency, and efficient record management.

✨ Key Features
👨‍💼 Admin Module
Secure login system (session-based)
Dashboard for managing records
Ability to store daily statistics:
Total inmates
Hospitalized residents
Staff count

💰 Donation System
Donors can submit name and amount
Each donation generates a unique QR ID
Stored securely in the database

📊 Daily Record Management
Tracks day-to-day operations
Automatically stores current date
Helps in monitoring facility status
